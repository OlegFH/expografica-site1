# Інструкція з розгортування сайту Expografica

## 📋 Передумови

- Хостинг з підтримкою PHP 7.4+
- Доступ до FTP або файлового менеджера хостингу
- Поддержка функції `mail()` для відправки email

## 🚀 Кроки розгортування

### 1. Підготовка файлів

1. Завантажте архів `expografica-site.zip`
2. Розпакуйте архів на локальному комп'ютері
3. Перевірте структуру папок:
   ```
   expografica-site/
   ├── index.html
   ├── css/styles.css
   ├── js/script.js
   ├── send.php
   ├── .htaccess
   ├── robots.txt
   ├── sitemap.xml
   ├── images/
   └── README.md
   ```

### 2. Завантаження на хостинг через FTP

1. Відкрийте FTP клієнт (FileZilla, WinSCP тощо)
2. Підключіться до хостингу:
   - Хост: `ftp.ваш-домен.com`
   - Користувач: ваше ім'я користувача FTP
   - Пароль: ваш пароль FTP
   - Порт: 21

3. Перейдіть до папки `public_html` або `www`

4. Завантажте всі файли з папки `expografica-site/`:
   - Завантажте HTML, CSS, JS файли
   - Завантажте папку `images/` з усіма зображеннями
   - Завантажте файли `.htaccess`, `robots.txt`, `sitemap.xml`
   - Завантажте файл `send.php`

### 3. Налаштування прав доступу (CHMOD)

1. Встановіть права доступу:
   - HTML, CSS, JS, PHP файли: **644**
   - Папки: **755**

2. У FileZilla:
   - Клацніть правою кнопкою на файл
   - Виберіть "File permissions"
   - Встановіть потрібні права

### 4. Налаштування email

1. Відкрийте файл `send.php` через текстовий редактор
2. Знайдіть рядок:
   ```php
   $to = 'expografica25@gmail.com';
   ```
3. Замініть на адресу, на яку хочете отримувати заявки

4. Збережіть файл та завантажте на хостинг

### 5. Налаштування домену

1. Логіньтесь у панель керування хостингом
2. Встановіть домен на папку, де розташований сайт
3. Чекайте поширення DNS (може зайняти до 24 годин)

### 6. Тестування

1. Відкрийте сайт у браузері: `https://ваш-домен.com`
2. Перевірте:
   - ✓ Сайт завантажується коректно
   - ✓ Усі зображення видно
   - ✓ Меню працює на мобільних пристроях
   - ✓ Форма замовлення відправляє email
   - ✓ Плавна прокрутка працює

### 7. SEO налаштування

1. Додайте сайт у Google Search Console
2. Додайте сайт у Yandex Webmaster
3. Завантажте `sitemap.xml` у пошукові системи
4. Перевірте, що `robots.txt` доступний

## 🔧 Налаштування SMTP (якщо mail() не працює)

Деякі хостинги не дозволяють використовувати `mail()`. У цьому випадку:

1. Попросіть у хостера дані SMTP сервера
2. Встановіть бібліотеку PHPMailer або SwiftMailer
3. Оновіть файл `send.php` для використання SMTP

Приклад з PHPMailer:
```php
<?php
require 'vendor/autoload.php';
use PHPMailer\PHPMailer\PHPMailer;

$mail = new PHPMailer(true);
$mail->isSMTP();
$mail->Host = 'smtp.gmail.com';
$mail->SMTPAuth = true;
$mail->Username = 'ваш-email@gmail.com';
$mail->Password = 'ваш-пароль';
$mail->SMTPSecure = PHPMailer::ENCRYPTION_STARTTLS;
$mail->Port = 587;
// ... решта коду
?>
```

## 🆘 Вирішення проблем

### Форма не відправляє email

1. Перевірте, чи ввімкнена функція `mail()` на хостингу
2. Перевірте логи помилок PHP
3. Спробуйте використовувати SMTP
4. Зв'яжіться з технічною підтримкою хостера

### Зображення не завантажуються

1. Перевірте, чи папка `images/` завантажена
2. Перевірте права доступу до папки (755)
3. Перевірте шляхи до зображень у HTML

### Сайт завантажується повільно

1. Оптимізуйте розмір зображень
2. Ввімкніть GZIP стиснення
3. Встановіть кеш браузера
4. Розглядайте використання CDN

### Меню не працює на мобільних

1. Перевірте, чи завантажений файл `js/script.js`
2. Перевірте консоль браузера на помилки
3. Очистіть кеш браузера
