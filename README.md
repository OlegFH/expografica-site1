## 🔧 Встановлення на хостинг

### Вимоги:
- PHP 7.4 або вище
- Поддержка функції `mail()` для відправки email

### Кроки:

1. **Завантажте файли на хостинг через FTP:**
   - Завантажте всі файли з папки `expografica-site/` на хостинг
   - Збережіть структуру папок (css/, js/, images/)

2. **Налаштування прав доступу:**
   - Встановіть права 644 для HTML, CSS, JS файлів
   - Встановіть права 755 для папок

3. **Налаштування email:**
   - Переконайтесь, що на хостингу ввімкнена функція `mail()`
   - Деякі хостинги вимагають налаштування SMTP

4. **Перевірка:**
   - Відкрийте сайт у браузері
   - Протестуйте форму замовлення

## 📧 Налаштування email

Файл `send.php` відправляє email на адресу: `expografica25@gmail.com`

## 🔐 Безпека

- Форма валідує всі вхідні дані
